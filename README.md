# Movyo Admin SaaS Dashboard Premium

Dashboard React/Vite para gestão completa do SaaS Movyo.

## Rodar local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Módulos incluídos
- Visão geral executiva
- Cadastro e gestão de restaurantes
- Planos e assinaturas
- Pedidos
- Pagamentos
- Bots WhatsApp
- Relatórios
- Segurança
- Configurações SaaS
